## Airbnb Hospitality & Booking Trends 

## Project Overview 
- This project focuses strictly on relational logic using JOINs and aggregation functions. 
- Subqueries and CTEs are intentionally excluded to strengthen core SQL fundamentals.

## Problem Statement: 
- Provide strategic insights into property performance and booking behavior. 

## Key SQL Tasks: 
- Host Success – Superhosts and total bookings received. 
- Property Revenue – Revenue by property type. 
- Guest Demographics – Average booking price by age group. 
- Booking Origins – Total bookings by nationality. 
- Rating Analysis – Properties with average rating above 4.5. 

## Core Concepts Covered: 
- Fact and dimension table joins 
- Aggregations with GROUP BY 
- Conditional filtering with WHERE 
- Sorting using ORDER BY

## Strengths: 
- Excellent focus on core SQL (JOINs + Aggregations). 
- Real-world business scenarios (Retail + Hospitality). 
- Proper reporting mindset with ORDER BY and aliases. 
- Good balance between difficulty and clarity.
